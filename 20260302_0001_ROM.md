```aiignore
1. 기억장치 분류 	
- 주기억장치(main memory)
	- CPU 에서 주소로 직접 접근 가능한 기억장치 
	- ROM, RAM 으로 구성 
- 보조기억장치(secondary storage) 
	- CPU 에서 주소로 직접 접근 불가 
		- IO 컨트롤러 경유해서 접근해야 함 
	- HDD, SSD 등 


2. 메모리 
- 관례적으로 주기억장치용도로 사용되는 기억장치를 메모리 라고 함 
	- 시대적으로 
		- 페라이트 코어로 직조
		- 집적회로(IC)로 구현

3. 종류
- 휘발성
	- RAM(random access memory)
		- SRAM
			- 주기적 리프레시 불필요	-> 빠르다.
			- 6T 필요	-> 구성 복잡하고, 비싸다.
		- DRAM
			- 주기적 리프레시 필요	-> 느리지만
			- 1T 1C	-> 상대적으로 저렴하다.
- 비휘발성 
	- ROM(read only memory)
		- 초기에는 기판에 소자로 ROM 기능을 기판으로 구현 		
		- Masked ROM 	-> 데이터를 제조시 입력 
		- PROM			-> 1회 기록 가능 
		- EPROM			-> 자외선 쬐서 지우고, 다시 기록 가능 
		- EEPROM 		-> 프로그램적으로 지우고, 다시 기록 가능 
	- 플래시 메모리 
		- NOR 			-> byte 단위 IO 지원, XIP(Execute In Place, 주메모리 복사 없이 바로 실행,  CPU 주소에 매핑가능해서 가능) -> 메인보드 BIOS 칩 
		- NAND 			-> 블록 IO, 집적용이 -> SSD 
```
